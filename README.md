  - Twitter (X)
  - Pinterest
  - Soundcloud
- YouTube downloading

# Usage

## Media Downloading

### Group Chat

1. Add @UMMRobot to the group chat
2. Send any supported social media post link to the chat

### Any Chat

1. Type `@UMMRobot` in any chat and paste the link to any supported social media
   post link (separated by space)
2. Press "Share post"

## YouTube Downloading

1. Add @UMMRobot to the group chat
2. Use `/audio link`, `/video link` or just `/audio`, `/video` **in reply** to
   the message containing a link
